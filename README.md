# BoosterVO - Site vitrine

Site Next.js 14 one-page pour BoosterVO, service d'audit de performance pour negociants VO.

## Installation

```bash
npm install
```

## Developpement

```bash
npm run dev
```

Le site est accessible sur [http://localhost:3000](http://localhost:3000).

## Build production

```bash
npm run build
npm start
```

## Logo

Placez votre logo dans `public/logo.jpg`.

## Deploiement

### Vercel

1. Connectez votre repo GitHub a Vercel
2. Le fichier `vercel.json` est deja configure
3. Vercel detecte automatiquement Next.js

### Netlify

1. Connectez votre repo GitHub a Netlify
2. Installez le plugin `@netlify/plugin-nextjs`
3. Le fichier `netlify.toml` est deja configure

## Stack technique

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Google Fonts (Nunito, Open Sans)
